Shower-MDX

Shower presentation generator with support of MDX syntax, powered by Next.js

Shower-MDX template

Shower Presentation Template is built by Vadim Makeev.

What is it?

This is a project template which lets you create your beautiful Shower presentations with markdown syntax and custom React components.

The following slides demonstrate rendered HTML. Check index.mdx to see how it looks in markdown.
